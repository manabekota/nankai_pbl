## Getting Started

#### 環境変数にAPIキーを設定
```
export OPENAI_API_KEY="YOUR KEY"
```

#### Streamlitのインストール
※Python環境が整っていることを前提とします。
```
# install
pip install streamlit

# 正常にインストールされたことの確認
streamlit hello
```
#### Langchainのインストール
```
pip install langchain
```
